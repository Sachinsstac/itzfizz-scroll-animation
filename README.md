# 🚀 ItzFizz Scroll Animation Portfolio

A modern and interactive web portfolio showcasing smooth scroll-based animations and real-world projects. Built with performance-focused design and clean UI.

---

## ✨ Features

- Smooth scroll animations using GSAP
- Modern dark-themed UI design
- Interactive project showcase section
- Clean and responsive layout
- Lightweight and optimized performance

---

## 🧩 Projects Included

### 🎬 CineMate X  
Movie recommendation system using machine learning and TMDB API.

### ⛳ Golf Subscription Platform  
Full-stack subscription-based platform built using EJS.

### 🚀 PitchKaro Landing Page  
Modern startup landing page with responsive and clean UI design.
